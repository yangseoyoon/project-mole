POINT INDEX

A Coordinate-Based Facial Classification System

Mole과 Point

해당 프로젝트는 신체상의 점(mole)과 기하학의 점(point)가
동음이의어라는 데에서 시작했다. 얼굴은 하나의 지도이고 점은
지도 위에 찍힌 표식이다. 사람마다 고유한 표식을 가지고 태어난다.
점은 단순히 신체의 흔적이 아니라 인생을 나타내는 좌표이고 당신의 삶을 대변하는 기호이다.
